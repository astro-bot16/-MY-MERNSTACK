# Bella Vista Restaurant Website

A responsive restaurant website built using HTML5 and CSS3. The website showcases Bella Vista, an authentic Italian restaurant, featuring sections for menu, about, contact information, opening hours, and social media links.

## Features

* Responsive navigation bar
* Hero section with restaurant introduction
* About Us section
* Menu showcase
* Contact section
* Responsive footer
* Smooth hover effects and transitions
* Mobile-friendly layout

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Responsive Design

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/bella-vista.git
```

2. Navigate to the project folder:

```bash
cd bella-vista
```

3. Open `index.html` in your browser or run it using VS Code Live Server.

## Sections

### Home

Introduces visitors to Bella Vista with an attractive hero section.

### About Us

Provides information about the restaurant and its history.

### Menu

Displays featured Italian dishes and specialties.

### Contact

Allows customers to view contact information and reach the restaurant.

### Footer

Contains quick links, opening hours, social media links, and copyright information.
