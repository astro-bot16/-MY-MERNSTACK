# 🌡️ Temperature Converter

A simple and responsive **Temperature Converter** built using **HTML, CSS, and JavaScript**. This application allows users to instantly convert temperatures between **Celsius (°C), Fahrenheit (°F), and Kelvin (K)** with real-time calculations.

---

## 🚀 Features

* 🌡️ Convert between:

  * Celsius (°C)
  * Fahrenheit (°F)
  * Kelvin (K)
* ⚡ Real-time temperature conversion while typing
* 🧹 Clear button to reset all input fields
* 📱 Fully responsive and clean user interface
* 🎨 Modern card-based design with hover effects
* 🔢 Values rounded to two decimal places for better readability

## 📐 Temperature Conversion Formulas

### Celsius to Fahrenheit

```
°F = (°C × 9/5) + 32
```

### Celsius to Kelvin

```
K = °C + 273.15
```

### Fahrenheit to Celsius

```
°C = (°F − 32) × 5/9
```

### Kelvin to Celsius

```
°C = K − 273.15
```

### Kelvin to Fahrenheit

```
°F = (K − 273.15) × 9/5 + 32
```
