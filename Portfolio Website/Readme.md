# 💼 Personal Portfolio Website

A modern, responsive **Personal Portfolio Website** built using **HTML, CSS, and JavaScript**. This project showcases my skills, projects, education, and contact information through a clean and interactive user interface.

---

## 🚀 Features

* 🏠 Responsive Hero Section
* 👨‍💻 About Me section with interactive tabs
* 🛠️ Skills and Education showcase
* 📂 Project Showcase
* 📱 Fully Responsive Design
* ✨ Smooth Scrolling Navigation
* ⌨️ Typing Animation using Typed.js
* 🔗 GitHub & LinkedIn Integration
* 🎨 Modern White and Orange Theme
* 📧 Contact Footer

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Typed.js
* Font Awesome
* Google Fonts (Poppins)
